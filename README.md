# sevenLanguagesInSevenWeeks

「[７つの言語 ７つの世界](https://www.ohmsha.co.jp/book/9784274068577/)」勉強用リポジトリ

## 対象言語

### Ruby

Homebrew でインストール

```
$ brew install ruby
```

ruby コマンドでファイルを実行

```
ruby sample.rb
```

### Prolog

Homebrew でインストール（SWI Prolog）

```
$ brew install swipl
```

swipl コマンドで実行

```
swipl sample.pr
```

halt.で閉じる

- Erlang
